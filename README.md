# CERS – React simple

Projet React simple pour la visibilité du Centre d’Étude et de Recherche du Sahel (CERS).

## Utilisation

1. Installer les dépendances :

```bash
npm install
```

2. Lancer le serveur de développement :

```bash
npm run dev
```

3. Construire la version de production :

```bash
npm run build
```

Le logo est dans `src/assets/logo.png` et les styles dans `src/styles.css`.
